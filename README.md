# DecodeLabs Data Science Internship

## Task 1: Dataset Understanding

### Objective
To understand the Spotify dataset and analyze its structure.

### Dataset Information
The dataset contains information about Spotify tracks including audio features and popularity metrics.

### Steps Performed
- Loaded dataset using Pandas
- Examined first 5 rows using head()
- Checked dataset structure using info()
- Analyzed dimensions using shape
- Generated statistical summary using describe()
- Reviewed column names and data types

### Tools Used
- Python
- Pandas
- Google Colab

### Outcome
Successfully understood the dataset structure, features, and data types, which forms the foundation for further analysis.
